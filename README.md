Installation
------------
```
composer require dutta/ovoid-api
```

Features
------------
- [x] login2FA
- [x] login2FAVerify
- [x] loginSecurityCode
- [x] verifyOVOMember
- [x] walletInquiry
- [x] getAccountNo
- [x] getAccountBalance
- [x] getOvoPoint
- [x] getBankList
- [x] transactionHistory
- [x] generateSignature `private` (for transfer more than 2 times)
- [x] transferOvo
- [x] transferBank
